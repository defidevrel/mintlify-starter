# Polymarket Rust SDK — Documentation

Mintlify docs for [polymarket-rs-sdk](https://github.com/defidevrel/polymarket-rs-sdk).

**Live site:** https://polymarket-rs.mintlify.app

## Edit docs

Content lives in `polymarket-sdk/*.mdx` and `docs.json`. Push to `main` to redeploy.

## Local preview

```bash
npm i -g mint
cd polymarket-rs-docs
mint dev
```

Open http://localhost:3000

## Repo

This repository (`polymarket-rs-docs`) is the source of truth for the public Mintlify deployment at [polymarket-rs.mintlify.app](https://polymarket-rs.mintlify.app).
